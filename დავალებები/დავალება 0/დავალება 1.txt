

# ­ЪЊў рЃџрЃћрЃЦрЃфрЃўрЃљ 0 РђЊ Git-рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃБрЃџрЃў рЃљрЃќрЃарЃЮрЃЋрЃюрЃћрЃЉрЃљ

_рЃАрЃарЃБрЃџрЃў рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌрЃљрЃЌрЃЋрЃўрЃА. рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ РђЊ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃцрЃўрЃЦрЃарЃў рЃЊрЃљ Git-рЃўрЃА рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ._

---


## ­Ъћ╣ рЃюрЃљрЃгрЃўрЃџрЃў A Рђћ Git-рЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃћрЃЉрЃў

### РюЁ A1. рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃарЃћрЃърЃЮ, рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЎрЃЮрЃЏрЃўрЃбрЃў
- [ ] `рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃў/рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 0` рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ VS Code-рЃерЃў
- [ ] рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃцрЃљрЃўрЃџрЃў `hello.txt` рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ: `Hello Git!`
- [ ] рЃЊрЃљрЃљрЃАрЃбрЃћрЃўрЃ»рЃћрЃЌ Рєњ рЃЎрЃЮрЃЏрЃўрЃбрЃў Рєњ Push
- [ ] рЃЌрЃБ рЃЎрЃЮрЃЏрЃўрЃбрЃА рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉ github desktop-рЃўрЃЌ рЃЏрЃљрЃерЃўрЃю рЃАрЃљрЃЌрЃљрЃБрЃарЃАрЃў рЃЕрЃљрЃгрЃћрЃарЃћ `рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 0` рЃљрЃЏ рЃЊрЃљ рЃЦрЃЋрЃћрЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃерЃў рЃЊрЃљ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃћрЃџрЃЉрЃљрЃАрЃў рЃарЃљрЃЏрЃћ рЃарЃљрЃАрЃљрЃф рЃерЃћрЃю рЃЏрЃЮрЃўрЃцрЃўрЃЦрЃарЃћрЃЉ:
- [ ] рЃЌрЃБ VS Code -рЃўрЃЌ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃЏрЃљрЃюрЃЊ 2 рЃЋрЃћрЃџрЃў рЃљрЃарЃљрЃљ 1 рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃЋрЃћрЃџрЃўрЃљ рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃўрЃЦ рЃЕрЃљрЃгрЃћрЃарЃћ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 0 - рЃўрЃЌ рЃЊрЃљрЃўрЃгрЃДрЃћ рЃЏрЃљрЃњрЃљрЃџрЃўрЃбрЃЊ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 0 - рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃбрЃћрЃАрЃб рЃцрЃљрЃўрЃџрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ
- [ ] рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ рЃцрЃљрЃўрЃџрЃў рЃЕрЃљрЃюрЃА GitHub-рЃќрЃћ

---


### ­Ъћё A2. рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў Рєћ рЃарЃћрЃЏрЃЮрЃБрЃЌ рЃАрЃўрЃюрЃЦрЃў
- [ ] рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ `hello.txt` рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ Рєњ Commit & Push
- [ ] GitHub-рЃќрЃћ рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ рЃўрЃњрЃўрЃЋрЃћ рЃцрЃљрЃўрЃџрЃў Рєњ Save (рЃљрЃЏ рЃюрЃљрЃгрЃўрЃџрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃўрЃЌрЃ«рЃЮрЃЋрЃЮ)
- [ ] рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ `git pull`
- [ ] рЃЊрЃљрЃљрЃЎрЃЋрЃўрЃарЃЊрЃўрЃЌ рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљрЃА

---


### ­ЪЊЦ A3. Fetch рЃЊрЃљ Pull рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ
- `git fetch` = рЃарЃћрЃЏрЃЮрЃБрЃЌрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ рЃерЃћрЃарЃгрЃДрЃЏрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ
- `git pull` = рЃЕрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљ рЃЊрЃљ рЃерЃћрЃарЃгрЃДрЃЏрЃљ

---


## ­ЪДЕ рЃюрЃљрЃгрЃўрЃџрЃў B Рђћ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃБрЃџрЃў рЃљрЃќрЃарЃЮрЃЋрЃюрЃћрЃЉрЃљ (рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ)

­ЪњА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃА рЃљрЃЦрЃЋрЃА: **рЃАрЃўрЃБрЃЪрЃћрЃбрЃў Рєњ рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў Рєњ рЃцрЃўрЃЦрЃарЃўрЃА рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў Рєњ рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ**

---


### ­ЪЈа B1. РђърЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃЮрЃЌрЃљрЃ«рЃўрЃљ?Рђю Рђћ рЃерЃБрЃЦрЃћрЃЉрЃўрЃЌ

рЃерЃћрЃюрЃЮрЃЉрЃљрЃерЃў рЃ«рЃљрЃарЃЌ. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЮрЃЌрЃљрЃ«рЃерЃў рЃерЃБрЃЦрЃў рЃљрЃю рЃљрЃюрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ, рЃљрЃю рЃЕрЃљрЃЦрЃарЃЮрЃЉрЃўрЃџрЃў. рЃЮрЃЌрЃљрЃ«рЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЎрЃљрЃарЃћрЃЉрЃўрЃЌ. рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃљрЃЊрЃњрЃћрЃюрЃЌ рЃ»рЃљрЃЏрЃерЃў рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃЮрЃЌрЃљрЃ«рЃўрЃљ?

**рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў:**
- рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃарЃБрЃЎрЃљ рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ
- рЃќрЃЮрЃњ рЃЮрЃЌрЃљрЃ«рЃерЃў рЃЉрЃюрЃћрЃџрЃљ

---


### ­Ъїђ B2. рЃњрЃљрЃЏрЃЮрЃАрЃљрЃАрЃЋрЃџрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ рЃџрЃљрЃЉрЃўрЃарЃўрЃюрЃЌрЃерЃў РђърЃЎрЃћрЃЊрЃћрЃџрЃќрЃћ рЃ«рЃћрЃџрЃўРђю

рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћ, рЃарЃЮрЃЏ рЃљрЃдрЃЏрЃЮрЃЕрЃюрЃЊрЃў рЃџрЃљрЃЉрЃўрЃарЃўрЃюрЃЌрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃДрЃЋрЃћрЃџрЃљ рЃЉрЃўрЃџрЃўрЃЎрЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃюрЃљрЃљ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЊрЃљ рЃЎрЃћрЃЊрЃџрЃћрЃЉрЃў рЃБрЃгрЃДрЃЋрЃћрЃбрЃљрЃЊ рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ. 
рЃерЃћрЃю рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЏрЃљрЃарЃ»рЃЋрЃћрЃюрЃљ рЃ«рЃћрЃџрЃў рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃћрЃарЃЌ рЃЎрЃћрЃЊрЃћрЃџрЃќрЃћ рЃњрЃЦрЃЮрЃюрЃЊрЃћрЃА рЃЏрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЊрЃљ рЃљрЃАрЃћ рЃўрЃљрЃарЃЮ. рЃљрЃ«рЃАрЃћрЃюрЃў, рЃарЃљрЃбрЃЮрЃЏ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃћрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃў рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃЮрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃўрЃА рЃърЃЮрЃЋрЃюрЃљрЃерЃў (рЃљрЃю рЃўрЃАрЃћрЃЋ рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃў рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљрЃерЃў). 
рЃљрЃАрЃћрЃЋрЃћ рЃЊрЃљрЃцрЃўрЃЦрЃарЃЊрЃў, рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ рЃўрЃАрЃћрЃЌрЃў рЃџрЃљрЃЉрЃўрЃарЃўрЃюрЃЌрЃў, рЃАрЃљрЃЊрЃљрЃф рЃћрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃў рЃЋрЃћрЃа рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃЊрЃљ рЃарЃљрЃбрЃЮрЃЏ.

**рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ:** рЃўрЃцрЃўрЃЦрЃарЃћрЃЌ рЃўрЃЏрЃљрЃќрЃћ, рЃарЃЮрЃЏ рЃ«рЃћрЃџрЃўрЃА рЃЎрЃћрЃЊрЃћрЃџрЃќрЃћ рЃЏрЃўрЃДрЃЮрЃџрЃўрЃЌ рЃћрЃарЃЌрЃАрЃљ рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃАрЃљрЃќрЃдрЃЋрЃљрЃарЃА рЃЏрЃўрЃАрЃЊрЃћрЃЋрЃЌ.

---


### ­ЪЊќ B3. рЃАрЃљрЃ«рЃћрЃџрЃў рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџ рЃАрЃўрЃљрЃерЃў

рЃњрЃљрЃЦрЃЋрЃЌ рЃњрЃарЃФрЃћрЃџрЃў рЃАрЃбрЃБрЃЏрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃљ рЃљрЃюрЃЉрЃљрЃюрЃБрЃарЃљрЃЊ рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў. рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃАрЃгрЃарЃљрЃцрЃў рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃљ рЃарЃљ рЃљрЃарЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃАрЃљрЃърЃЮрЃЋрЃюрЃћрЃџрЃљрЃЊ? рЃљрЃдрЃгрЃћрЃарЃћрЃЌ рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў, рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.

**рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ:** рЃљрЃа рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃДрЃЋрЃћ рЃАрЃљрЃЌрЃўрЃЌрЃљрЃЮрЃЊ.

### ­ЪЋњ B4. рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃЉрЃћрЃЋрЃарЃў рЃерЃћрЃ«рЃЋрЃћрЃЊрЃарЃљ

### Рџќ№ИЈ B5. рЃАрЃљрЃАрЃгрЃЮрЃарЃўрЃА рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ (рЃДрЃљрЃџрЃЉрЃў рЃЏрЃЮрЃюрЃћрЃбрЃљ)

### ­ЪЊџ B6. рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃўрЃА рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ

### ­ЪЊЉ B7. рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃбрЃћрЃЉрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ рЃерЃћрЃќрЃдрЃБрЃЊрЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃЋрЃћрЃЉрЃўрЃЌ

рЃњрЃљрЃЦрЃЋрЃЌ рЃЉрЃћрЃЋрЃарЃў рЃцрЃЮрЃарЃЏрЃљ, рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃА рЃљрЃЦрЃЋрЃА рЃюрЃЮрЃЏрЃћрЃарЃў. рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃАрЃбрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃ«рЃћрЃџ рЃњрЃљрЃЊрЃљрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃЮрЃЌ рЃЊрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃюрЃЮрЃЏрЃћрЃарЃў рЃЊрЃљрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃЮрЃЌ рЃљрЃю рЃЕрЃљрЃўрЃюрЃўрЃерЃюрЃЮрЃЌ. рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃљрЃарЃЎрЃЋрЃћрЃЋрЃЌ, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃюрЃЮрЃЏрЃћрЃарЃў рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃћрЃЉрЃБрЃџрЃў?

### ­Ъџц B8. рЃЏрЃЊрЃўрЃюрЃљрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃЎрЃЋрЃћрЃЌрЃљ
